🧱 Breakout – Консольная версия на 7 языках
Классическая аркада «Breakout» (наследник Arkanoid) прямо в вашем терминале!
Управляйте платформой, отбивайте мяч, разрушайте блоки и проходите уровни.
Реализована на семи языках программирования с единым геймплеем, цветной графикой, системой очков и рекордов, бонусами и настраиваемой сложностью.

🚀 Возможности
Игровой процесс – управляйте платформой с помощью стрелок, отбивайте мяч, разрушайте блоки.

Блоки – цветные блоки с разной прочностью (от 1 до 3 попаданий).

Бонусы – при разрушении блоков иногда выпадают бонусы:

🟢 Широкая платформа – увеличивает ширину платформы.

🔵 Ускорение мяча – временное увеличение скорости.

🟡 Дополнительная жизнь – +1 жизнь.

🔴 Огненный мяч – прожигает блоки насквозь.

Уровни – после очистки поля от блоков открывается следующий уровень с новой конфигурацией.

Счёт и рекорд – текущий счёт и лучший результат сохраняются в локальный файл.

Цветная графика – платформа, мяч, блоки и бонусы окрашены.

Управление – стрелки влево/вправо для движения, пробел для запуска мяча, Q для выхода, R для перезапуска.

Кроссплатформенность – работает в любом терминале с поддержкой ANSI (Linux, macOS, Windows с WSL или ConEmu).

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [опции]
Опции
Опция	Описание
-s, --speed <N>	Базовая скорость игры (задержка в мс, по умолчанию 50)
-h, --help	Справка
Управление
Клавиша	Действие
← / A	Движение платформы влево
→ / D	Движение платформы вправо
Пробел	Запуск мяча / пауза
R	Перезапуск после окончания игры
Q	Выход из игры

Python
bash
pip install windows-curses  # только для Windows
python breakout.py [опции]
C++
bash
g++ -std=c++17 breakout.cpp -o breakout -lncurses
./breakout [опции]
Требуется библиотека ncurses (установите через пакетный менеджер).

java -cp .:lanterna.jar breakout [опции]
Требуется библиотека lanterna (скачайте JAR и добавьте в classpath).

🧠 Логика игры
Инициализация – платформа внизу, мяч на платформе, блоки выстроены в верхней части экрана.

Запуск мяча – пробел отрывает мяч от платформы; мяч летит под углом.

Отскоки – мяч отскакивает от стен, потолка и платформы; угол отскока зависит от точки касания платформы.

Разрушение блоков – при попадании мяча в блок он теряет прочность; при нулевой прочности блок исчезает (+очки).

Бонусы – при разрушении блока с вероятностью 15% выпадает бонус, который падает вниз; при касании платформы активируется.

Уровни – после разрушения всех блоков генерируется новый уровень с увеличенной скоростью.

Поражение – если мяч падает вниз, теряется жизнь; при 0 жизней игра заканчивается.

Счёт и рекорд – сохраняются в файл.

✨ Дополнительные фичи
Анимированный мяч – меняет цвет при отскоке.

Звуковые эффекты – при ударе о стену, блок и бонус (ASCII bell).

Пауза – нажмите P для приостановки.

Автосохранение – рекорд обновляется автоматически.
